第一次翻轉新聞松來了！ / 2017/04/16

想用新科技解決當前的新聞難題嗎？

想通過科技方式加快新聞產製流程？

想知道參與黑客松對新聞有什麼幫助嗎？

4月16日（星期六），Hacks/Hackers Taipei 將聯合 g0v.news 共同舉辦翻轉新聞松（Hacking Journalism），邀請記者、編輯、設計師、前後端工程師，透過一天的活動，一起動腦、一起實作，跨界打造能實際解決當前新聞問題的報導或新聞產品。

現在就報名！

活動時間

2017年4月16日(六) (加到我的 Google Calendar)

活動地點

TAF 空總創新基地「士官大樓」3F ( 台北市大安區建國南路一段177號 / 詳細交通資訊 / 感謝極客窩提供場地 )

適合參與對象

記者、編輯、前後端工程師、設計師、插畫家、創業家、新聞傳播學院師生、或任何關心新聞媒體發展的朋友，你可能有想做的報導，但缺乏跨界幫忙；也可能你有好的科技點子，但希望讓更多新聞人認識；又或是你只是單純好奇黑客松是什麼？黑客松跟新聞人到底有什麼關係？通通歡迎參加！


報名方式

4月10日（星期日）開放網路報名：www.accupass.com/go/hhtpexg0vnews

  • 票券種類：分1)記者編輯、2)前後端工程師、3)設計師三種票，請依照您個人專長或興趣選擇合適的票種報名。
  • 票券費用：0元，但報到當天需繳交新台幣150元的午餐費用（為了方便大家邊吃邊討論，預計提供能一手拿的食物。素食者煩請報名時註明）。

活動方式

帶著問題和方向來參與：先上網（http://beta.hackfoldr.org/news-hackathon）登記你的提案，若事先沒有計畫的人，也能到了現場再分組。

可單槍匹馬或組隊參加：一個完整的科技方案，少不了新聞人、科技人、設計師等多元協作。你可以帶著點子單槍匹馬報名，現場招兵買馬，或能事前在Hacks/Hackers Taipei粉絲團po文，號召參與者。

為鼓勵開源協作，本活動當天所產出的程式碼及內容，將以開源的授權條款 （MIT, CC 等）釋出。活動結束後所產生的內容，則不在此限。


活動流程

  • 11:00 開場、提案 (1個提案：3分鐘）
  • 11:40 自介、午餐、交流
  • 18:00 成果報告
  • 19:00 收播，閒聊，散會

如何製作沉浸式新聞？ / 2017/03/04

你看過VR（虛擬實境）或AR（擴增實境）新聞嗎？你好奇紐約時報、BBC等國外媒體怎麼運用Google Earth Pro、Google Street View等線上工具進行新聞報導嗎？當愈來愈多科技工具，能加強第一視角敘事、讓讀者站在第一線體驗新聞情境時，新聞人該如何應用最新工具，說一個好故事？

3月4日（六）我們將舉辦Hacks/Hackers Taipei的第三次小聚，特別邀請前路透記者、現任Google News Lab新聞媒體培訓師Irene Jay Liu，以及聯合報新媒體中心內容創新組組長蔡幸怡，一起來談談如何透過VR、地圖街景等工具進行創新報導。


活動細節

  • 時間：2016 年 3 月 4 日 13:30 ~ 17:00
  • 地點：Google Taipei 辦公室 ( 台北市信義區信義路五段7號 )

當天流程：

  • 13:30-14:00 報到
  • 14:00-14:10 Opening

  • 14:10-15:00 怎麼製作沉浸式新聞？/ Google News Lab, Irene Jay Liu (英語分享）

    除了介紹Youtube 360、 Cardboard、Google Street View、Google Earth Pro等新工具如何應用於新聞報導，Irene也將從個人經驗來談談如何在新聞編輯台導入創新基因。

  • 15:00-15:20 交流時間

  • 15:20-16:20 聯合報VR&AR說故事的n個心得 / 聯合報新媒體中心 內容創新組組長 蔡幸怡

    分享聯合報新媒體中心2016年在VR&AR說故事的初探，從實作中學到的心得（與苦惱）

  • 16:20-17:00 短講與交流

期待你當天一起加入！( 注意：這次場地管制較嚴需事先準備名單，請盡量確定當天一定可以出席再報名喔！ )

講師陣容

Irene Jay Liu / Google News Lab

曾經擔任路透社記者、香港南華早報特別專案負責人、美國哥倫比亞大學新聞學院講師，現任Google News Lab新聞媒體培訓師，也同時兼任香港大學、香港中文大學資料新聞課程程講師。

蔡幸怡 / 聯合報新媒體中心 內容創新組組長

媒體資歷六年，現任聯合報新媒體中心內容創新組組長。走過平面編輯室進到數位製作，身在老傳統與新方法之間，連結storytelling & tech ，帶領團隊實作聯合報數位創新說故事，包括資料新聞專案、多媒體專題等。

手把手的 Github 入門實戰 / 2016/12/3

你以為Github是工程師的臉書，跟新聞人無關？錯！有越來越多國外媒體透過 Github協作、找新聞、發表報導、與讀者互動、甚至超越公司組織在github上相互學習。

這次 Hacks/Hackers Taipei 請到來自零時政府社群的 Kirby Wu 為你帶來 120 分鐘的精彩實作課程，從 Git 的基本概念到實際建立一個屬於自己的個人網站。內容包括：

  • 什麼是「版本管理系統」？
  • Git 與 Github 簡介
  • 手把手的 Github 操作入門
  • 無腦網頁製作實戰 ( 附送免費子網域一組! )
  • Game Time ! - 利用 Github 進行分組對戰吧
